Rent It Forward: Visit site

Rent It forward is a site for the times when a user wants to buy something that you only need on a temporary basis. They list the item and other users can make offers to buy the item when the original user is done with it. This enables the original user to work with a brand new item and the secondary buyer to get a very gently used item for a good deal. It's a win for both users!

File Share Tool: Visit site

File Share Tool is an early rails app for me. A user can upload a file(which is stored on S3) and add a message. Then an email is sent to the recipient along with a link to access the file. When the recipient clicks on the link they have the ability to download the file and the original 'sharer' is sent a confirmation email that the recipient has viewed the link'

Stack: Rails, Postgres, AWS S3
